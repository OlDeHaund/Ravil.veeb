MVP mõiste

Minimal Viable Product - Тестовая версия продукта, услуги, товара, которая показывает жизнеспособность проекта.
Его делают, чтобы быстро проверить, нравится ли идея, и собрать мнение от пользователей.
Полезность разработки MVP доказывают примеры крупных на данный момент компаний. Например, Даниэль Эк и Мартин Лорентсон в 2006 году запустили небольшой сервис с одной функцией — потоковая передача музыки. Сейчас мы знаем этот продукт под названием Spotify.

Kaug-repository loomine

Sisselogida GitHubis → valida „New repository” → sisesta reposi nimi ja vali seaded (näiteks privaatsus) → loo repo ja kopeeri selle URL.

git remote add origin https://github.com/OlDeHaund/Ravil.veeb.git git add .
git commit -m -a "Tekst mis me paneme kommetreerimiseks"
git push -u origin master/main
